# Campaign Quality Checklist

Use this checklist to ensure your campaign is complete, consistent, and ready to run.

---

## Completeness Check

### Core Documents

- [ ] **campaign-overview.md** exists and is complete
  - [ ] Adventure background written
  - [ ] Adventure synopsis with major story beats
  - [ ] 2+ adventure hooks provided
  - [ ] All chapters listed with summaries
  - [ ] Climax and resolution described
  - [ ] Character advancement guidance provided

- [ ] **README.md** exists and is complete (player-facing)
  - [ ] Campaign overview (spoiler-free)
  - [ ] Essential information (level, length, tone)
  - [ ] Setting information
  - [ ] Character creation guidelines
  - [ ] Session Zero topics covered
  - [ ] No spoilers included

- [ ] **chapters-summary.md** exists and is complete (if campaign has multiple chapters)
  - [ ] Chapters and scenes listed

- [ ] **npcs.md** exists and contains all important NPCs
  - [ ] 3-5 major NPCs fully detailed
  - [ ] Supporting NPCs included
  - [ ] Stat blocks or references provided
  - [ ] Roleplaying notes included
  - [ ] Prompts for each NPC to aide image/character sheet creation

- [ ] **locations.md** exists and contains all key locations
  - [ ] 3-5 major locations fully detailed
  - [ ] Supporting locations included
  - [ ] Travel times and connections noted
  - [ ] Maps or descriptions provided
  - [ ] Prompts for each loaction to aide image creation

- [ ] **factions.md** exists (if campaign includes factions)
  - [ ] 2+ factions detailed
  - [ ] Goals and relationships defined
  - [ ] Reputation system explained

- [ ] **timeline.md** exists and is complete (if campaign has multiple chapters)
  - [ ] Chapters and scenes listed in chronological order

### Chapter Documents

For each chapter (chapter-01.md, chapter-02.md, etc.):

- [ ] Chapter overview section complete
  - [ ] Synopsis written
  - [ ] Objectives clearly defined
  - [ ] Key NPCs and locations listed

- [ ] 3-5 scenes per chapter
  - [ ] Each scene has read-aloud text
  - [ ] Each scene has DM information
  - [ ] Possible player actions considered
  - [ ] Encounters designed

- [ ] Chapter climax exists
  - [ ] Meaningful challenge
  - [ ] Multiple resolution paths
  - [ ] Consequences defined

- [ ] Chapter resolution
  - [ ] Transition to next chapter
  - [ ] XP or milestone guidance

- [ ] DM notes and tips section
  - [ ] Pacing guidance
  - [ ] Common pitfalls addressed
  - [ ] Contingency plans for problems

### Minimum Viable Campaign

At minimum, a campaign should have:
- [ ] 1+ complete chapter
- [ ] Campaign overview
- [ ] Briefing document
- [ ] Key NPCs documented
- [ ] Key locations documented
- [ ] Artwork prompts for all key NPCs and locations

---

## Consistency Check

### Names and References

- [ ] NPC names spelled consistently throughout all documents
- [ ] Location names spelled consistently
- [ ] Faction names spelled consistently
- [ ] Titles and roles consistent (e.g., "Captain" vs "Commander")

### Cross-References

- [ ] All links between documents work
  - [ ] Campaign overview links to chapters
  - [ ] Chapters link to NPCs document
  - [ ] Chapters link to locations document
  - [ ] Chapters link to each other
  - [ ] Artwork prompt documents exists for referenced artwork

- [ ] NPC references are accurate
  - [ ] NPCs appear in chapters where stated
  - [ ] NPC descriptions match between documents
  - [ ] Stat blocks referenced correctly

- [ ] Location references are accurate
  - [ ] Locations appear in chapters where stated
  - [ ] Descriptions match between documents
  - [ ] Travel connections make sense

### Timeline and Continuity

- [ ] Events happen in logical order
- [ ] Time passing makes sense
- [ ] No contradictions in timeline
- [ ] NPC knowledge is consistent
  - [ ] NPCs know what they should know
  - [ ] NPCs don't know things they couldn't know
  - [ ] Information spreads logically

- [ ] Story logic holds
  - [ ] No plot holes
  - [ ] Cause and effect makes sense
  - [ ] Motivations remain consistent

### Geographic Consistency

- [ ] Travel times are consistent
- [ ] Locations have logical relationships
- [ ] Maps match descriptions
- [ ] No impossible geography (unless intentional/magical)

---

## Balance Check

### Encounter Difficulty

For each chapter, check that:
- [ ] Combat encounters appropriate for party level
- [ ] Mix of easy, medium, and hard encounters
- [ ] Deadly encounters used sparingly and meaningfully
- [ ] Encounter variety (not all combat)

### Challenge Variety

- [ ] Mix of combat, social, and exploration challenges
- [ ] Different skills get to shine
- [ ] Different character types have moments
- [ ] Not too much of any one type of challenge

### Rewards and Progression

- [ ] Treasure appropriate for level
- [ ] Magic items distributed reasonably
- [ ] Progression pace feels right
  - [ ] Not too fast (players don't feel achievement)
  - [ ] Not too slow (players feel stuck)

- [ ] Rewards match effort
  - [ ] Big challenges have meaningful rewards
  - [ ] Rewards aren't just gold and items

### Difficulty Curve

- [ ] Campaign starts at appropriate difficulty
- [ ] Difficulty increases over time
- [ ] Climax is most challenging part
- [ ] No sudden difficulty spikes unless intentional

---

## Story Quality Check

### Player Agency

- [ ] Players have meaningful choices
- [ ] Multiple solutions to problems
- [ ] Player decisions have consequences
- [ ] Story isn't a railroad (forced single path)
- [ ] NPCs can be befriended, antagonized, or ignored

### Engagement

- [ ] Strong hooks that pull players in
- [ ] Personal stakes for characters
- [ ] Interesting NPCs with motivations
- [ ] Compelling villain or conflict
- [ ] Mystery or questions to answer

### Pacing

- [ ] Opening chapter grabs attention quickly
- [ ] Good mix of action and downtime
- [ ] Builds toward climax
- [ ] Doesn't drag in the middle
- [ ] Satisfying resolution

### Story Structure

- [ ] Clear beginning, middle, and end
- [ ] Rising action and tension
- [ ] Climactic moment
- [ ] Resolution that feels earned
- [ ] Hooks for future adventures (if desired)

---

## Usability Check

### At-the-Table Usability

- [ ] Read-aloud text is clear and evocative
- [ ] DM information is concise and scannable
- [ ] Important information stands out
- [ ] Stat blocks easy to reference
- [ ] Quick reference tables where helpful

### Formatting

- [ ] Consistent markdown formatting
- [ ] Headers used properly (# for title, ## for sections, etc.)
- [ ] Lists formatted correctly
- [ ] Tables formatted correctly
- [ ] Code blocks used appropriately for stat blocks
- [ ] Blockquotes used for read-aloud text

### DM Guidance

- [ ] Sufficient tips for running scenes
- [ ] Contingency plans for common problems
- [ ] NPCs have roleplaying notes
- [ ] Tactical guidance for complex encounters
- [ ] Pacing suggestions provided

### Organization

- [ ] Files named consistently
- [ ] Logical folder structure
- [ ] Documents easy to find
- [ ] Related content grouped together

---

## Tone and Style Check

### Consistency with Tone

- [ ] Content matches stated tone (heroic/dark/comedic/etc.)
- [ ] No jarring tonal shifts unless intentional
- [ ] Descriptions support the mood
- [ ] NPCs fit the tone

### Appropriate Content

- [ ] Content warnings in briefing if needed
- [ ] Mature content flagged
- [ ] Session Zero guidance for sensitive topics
- [ ] Content appropriate for intended audience

### Writing Quality

- [ ] Descriptions are vivid but not overly long
- [ ] Read-aloud text flows well when spoken
- [ ] No excessive jargon
- [ ] Grammar and spelling checked
- [ ] Consistent voice throughout

---

## Technical Check

### Stat Blocks

- [ ] All combat encounters have stat blocks or references
- [ ] Stat blocks are accurate
- [ ] References to Monster Manual or other books correct
- [ ] Custom monsters balanced for level
- [ ] Modifications to standard monsters noted

### DCs and Numbers

- [ ] Skill check DCs appropriate for level
- [ ] Save DCs appropriate for level
- [ ] Damage appropriate for level
- [ ] Treasure values reasonable
- [ ] XP totals correct (if using XP)

### Rules and Mechanics

- [ ] Uses official 5e rules correctly
- [ ] Homebrew rules clearly marked
- [ ] Variant rules explained
- [ ] No broken mechanics
- [ ] Special abilities explained

---

## Polish Check

### Image Prompts

- [ ] Key scenes have image prompts
- [ ] Important locations have image prompts
- [ ] Major NPCs have image prompts
- [ ] Prompts are detailed and useful
- [ ] Prompts include proper metadata

### Supporting Materials

- [ ] Handouts created where appropriate
- [ ] Custom magic items balanced
- [ ] Custom monsters playtested or carefully considered
- [ ] Random encounter tables if needed
- [ ] Helpful reference tables included

### Extra Mile

- [ ] Faction reputation system (if applicable)
- [ ] Timeline of events
- [ ] World map or regional maps
- [ ] NPC and location quick reference tables
- [ ] Session recap template

---

## Final Validation

### Read-Through Test

- [ ] Read entire campaign start to finish as DM
- [ ] Verify flow makes sense
- [ ] Check that information appears when needed
- [ ] Ensure no forward references to unknown info

### Logic Test

Ask yourself:
- [ ] Would this be fun to play?
- [ ] Would this be fun to run as DM?
- [ ] Do player choices matter?
- [ ] Are there memorable moments?
- [ ] Is the story satisfying?

### Completeness Test

- [ ] DM could run Session 1 with current materials
- [ ] All necessary information is present
- [ ] No missing sections or "TODO" markers
- [ ] Campaign is ready to play

**Scoring:**
- **25-30:** Excellent, ready to publish
- **20-24:** Good, ready to run with minor tweaks
- **15-19:** Needs some work in weak areas
- **10-14:** Significant revisions needed
- **Below 10:** Major problems to address

## Common Issues and Fixes

### Issue: Too Much Detail
**Fix:** Cut unnecessary descriptions, focus on what's useful at the table

### Issue: Not Enough Detail
**Fix:** Add DM notes, roleplaying tips, and contingency plans

### Issue: Railroad Plot
**Fix:** Add alternative paths, give players meaningful choices

### Issue: Inconsistent Names
**Fix:** Search and replace, create NPC reference list

### Issue: Broken Cross-References
**Fix:** Check all links, update chapter numbers if needed

### Issue: Difficulty Spikes
**Fix:** Smooth progression, add scaling guidance

### Issue: Boring Encounters
**Fix:** Add environmental features, interesting terrain, NPC motivations

### Issue: Unclear Objectives
**Fix:** Make goals explicit, give players clear direction

### Issue: Weak Climax
**Fix:** Build tension, raise stakes, ensure multiple resolution paths

### Issue: Unsatisfying Resolution
**Fix:** Show consequences, tie up threads, provide closure
