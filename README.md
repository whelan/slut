# Tyranny of Dragons – Agent Quickstart

Dette repository er et **markdown-only** DM-projekt for *Tyranny of Dragons* (D&D 5e 2024), med fokus på Well of Dragons og Temple of Tiamat.

## Start her (autoritativ reference)

1. **Læs først:** [`tyranny-of-dragons-kampagne.md`](tyranny-of-dragons-kampagne.md)  
   (kilden til sandheden for tone, regler, party-status og kampagnekontekst)
2. Brug derefter relevante undermapper nedenfor.
