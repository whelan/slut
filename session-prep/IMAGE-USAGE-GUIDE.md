## Session-by-Session Breakdown

### PRE-SESSION (Setup)

- [ ] Load all token images into VTT software (Foundry, Roll20, etc.)
- [ ] Place battlemap files in VTT battle-map folder
- [ ] Test image scaling on your display (should be clear at arm's distance)
- [ ] Print NPC Consequence Matrix for reference (if running offline)

### SESSION PHASE A – Caldera & Entry (60–90 min)

**Key Images to Display:**

1. **[Template Exterior Cinematic]** – Before plaza encounter
   - Read boxed text while players see the temple rising from ash and bone
   
2. **[Red Wizard Generic Token + Dragonfang/Dragonclaw tokens]** – If plaza combat erupts
   - Use for devils + guard drakes (or cult warriors if adaptation used)

3. **[Veksin Young Red Wizard Token]** – Blue Chapel entrance
   - Display as the party enters Area 1
   - Use for single-Red-Wizard encounter

### SESSION PHASE B – Level 2: The Fivefold Sanctum (60–90 min)

**Key Images to Display:**

1. **[Fivefold Sanctum Battlemap]** – Place on grid
   - Shows five chapels, portal at center, Abishai positions
   - Keep visible during all sabotage/combat encounters

2. **[Rath Modar Token]** – If he engages
   - Red Wizard master; ritual leader
   
3. **[Galvan, Rezmir, Neronvain Abishai Tokens]** – Color-specific encounters
   - Galvan (Blue) for Blue Foci
   - Rezmir (Black) for Black Foci
   - Neronvain (Green) for Green Foci

### SESSION PHASE C – Level 3: The Crown (90–120 min)

**Key Images to Display:**

1. **[Crown Dais Battlemap]** – Place on grid at start of combat
   - Severin at center, portal at back, five dragon-head openings around perimeter
   - Keep visible throughout all combat

2. **[Severin Masked Token]** – His figure on the dais
   - Display prominently; this is the final boss
   - Update his condition as HP drops through Phase 1 → Phase 2

3. **[Tiamat Manifestation Portal]** – As clock rises
   - Display at Clock 3 (voice only)
   - Update as heads press through: Clock 4 (1st head), Clock 6 (head attacks), Clock 7 (partially manifested), Clock 8 (fully manifested)
   - Use to show escalation visually

4. **[Elia Bronze Dragon Token]** – If triggered (Clock 4–5, Party HP < 150)
   - Dramatic moment; metallic ally arrival
   - Keep on screen during her combat round

### POST-COMBAT: Aftermath (30–60 min)

**Key Images to Display:**

1. **[Elia Bronze Dragon Token]** – Scene 6 (Full Victory only)
   - Shows her landing on caldera rim, thanking the party

2. **[Temple Exterior]** – Modified for aftermath (optional)
   - Collapsed temple, ash falling, sunrise breaking through

## VTT-Specific Notes

### Foundry VTT
- **Token size:** 1024×1024 px images → set as "1 unit = 1 grid square" (actors have 1 square footprint)
- **Battlemap placement:** 1024×1024 maps → set grid to 10×10 or 20×20 depending on scale
- **Portraits:** Display in character sheets or as "portrait" mode in chat

### Roll20
- **Token size:** 1024×1024 images → 70 px = 5 ft. scale
- **Battlemaps:** Upload directly to map layer; adjust fog of war as needed
- **Token library:** Pre-load all tokens into campaign library before session

### Offline / Physical Table
- **Print tokens:** 2"×2" or 3"×3" card stock, sleeve with clear sleeves (reusable)
- **Battlemaps:** Print on poster paper (11×17") or use tablet display
- **Cinematic shots:** Project on secondary monitor or TV behind the DM screen for atmosphere

## Image Storage & Naming

All session images are stored in `/art/finale/output/` with consistent naming:

```
[character-or-scene]-[role/type]-[date-time].png
```

Examples:
- `severin-masked-token_20260516_164330_1.png`
- `galvan-blue-abishai-token_20260516_164526_1.png`
- `temple-exterior-cinematic_20260516_161845_1.png`

**Backup:** Copy entire `/art/finale/output/` folder before session.
